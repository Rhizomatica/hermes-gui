Help

Login

Use your required credentials to access the HERMES system. e.g.

Username: hermes
Password: hermes1234

Your credentials were created and provided beforehand, most likely. If you have no access to this information or you forgot your password contact for the system administrator.

Voice

Use the voice panel to communicate via voice PTT (push to talk).

Step - Use the step button or select the number to easily change the frequency of the radio.

Volume - Volume control to regulate the speaker volume.

Reset Protection - If the radio turns the protection ON you can reset the protection selecting (Reset). This usually happens when the antenna does not match the transmission frequency.

Mode - You can change the operating mode (USB/LSB) by toggling the Mode option button.

Messages

List of public messages received.

Hermes system allows transmission of text and files between connected stations, through public messages, sent through this interface or private messages, though the Deltachat application or webmail interface.

On the Messages page you can see the public, sent messages. If you have system permissions you can send a new message to another station.

Sent messages

List of sent messages, outbox.

*Note: When a sent message appears in this list it does not mean the message was received by the destination or even transmitted. Check the transmission list to verify the transmission status of your sent messages.

Message

The message received detail allow you to see all message information: title, message, file, date and time, origin and destinations. Select the reply button to answer the message.

*Note: If the message is protected, type the correct password (defined previously by the emitter/origin) to access the message content.

New message

When composing a new message you can protect messages with a passcode before sending. Opening the message will require entering the same passcode. The receiver of the protected message must know the passcode to read the entire message and access the attached files.

Attach file

Attach a file to your message. It is possible to attach images, audio documents, etc.

Accepted formats: The system will accept all types of files. For images and audio files the size limit is 31.45 MB, for other kind of files the size limit is 20.48 kB.

*Note: The system will compress attachments, however, message packages greater than 71.68 MB will not be transmitted.

Email

On the Email page you can find the Webmail system, Deltachat downloads and the stations email list. You can also find emergency email contacts for unexpected or emergency situations.

*Note: To access the webmail you have to use the same credentials used to access the HERMES system.

Deltachat

Through the Deltachat app, you can use emails to conduct personal message exchange communications (similar to WhatsApp or SMS).

The HERMES system includes a compression system. End-to-end encryption should be disabled in order to allow the server-based image and audio compression to work properly, otherwise image and audio exchange will not be possible.

The steps to find this feature in Deltachat are: -> Advanced -> Autocrypt, Prefer End-To-End Encryption: turn off: If you wish, you can also turn on email delivery monitoring to check if your emails were correctly delivered.

Radio Configuration

The Radio Configuration page you can find all information about the Radio Analog and Digital transmission.

Radio Status - radio connection status (connected / disconnected).

Digital Data frequency - Frequency used to transmit messages and files. All stations need to be on the same frequency so they can communicate with each other.

*Note: Changing the frequency can stop or interrupt the sending and reception of transmissions.

SWR - Standing Wave Ratio. This indicates how efficiently power is transmitted from the radio transmitter to the antenna. A low SWR (close to 1) indicates that the antenna is set to the correct operating frequency and the system will operate efficiently. A high SWR suggests that the antenna is not set to the same frequency as the transmitter or that it is damaged. The transmission will not be optimal and trying to operate the system with a high SWR can damage the transmitter and will likely cause the radio to go into protection Mode.

Power - It is best practice to use the minimum power needed to make contact. Using a lower power setting can shorten the transmission range and preserve battery capacity.

Operation mode (TX/RX) - TX stands for transmitter and RX stands for receiver.

Serial Number - Identifies your radio equipments unique serial number.

Digital data frequency

Digital Data frequency for messages and files transmission.

Note: Changing the frequency can stop or interrupttheall sending and reception ofive transmissions.

Analog frequency

Analog frequency for voice usage.

*Note: This frequency is not related to the message and emails transmission and only applies to voice transmission and reception.

Threshold

Threshold of SWR allowed. If the SWR level goes above this threshold protection will be enabled and transmission will halt.

Mode

The radio can operate on two different modes USB / LSB. All stations need to be on the same mode so they can communicate with each other optimally.

Radio Profile

Choose the radio profile for operation. Phony for voice or digital for data transmission.

PTT (Push to talk)

Push To Talk. This function simulates pressing the PTT button on the physical microphone. Used only for testing purposes when tuning the system.

Reboot

Reboot the HERMES system.

Reset to default

Reset/restore your radio to the default HERMES configuration. You won't lose your messages.

*Note: Restoring your radio to the default configurations can stop or interrupt sending and reception of transmissions.

Reset protection

If the SWR goes above the threshold set on “Protection activation threshold level” this function will halt transmission to avoid damage to the equipment. Click reset to resume system operation, having first reviewed why the system went over the threshold.

User management

In User management you can update your own information (Phone number and password) or delete your user.

If you are an administrator you can create, update and delete all users with exception of the root user.

*Note: if you delete your user credentials you won't be able to access the HERMES system anymore.

Message Configuration

Manage which user profile has permission to send messages and attach files.

Broken files can consume your storage, clean broken files to free your storage by eliminating unnecessary files.

Transmission schedule

Manage your transmission schedules, by time period and station name. As communication between stations is unidirectional, transmissions must be scheduled to happen according to the central station, alternating between reception and transmission with different stations at different times. Keep in mind that due to the scheduling of transmission of the messages, they may take a while to arrive, depending on the amount of messages in the queue and the gateway station's schedule of operation.

Transmission list

When you press send on a message, it is a transmission queue before its final destination. A system administrator can cancel the messages in the queue before they are sent, and also delete the sent and received messages. Deleted sent and received messages periodically can help avoid the hard drive from filling up.

In an emergency, the system administrator can force message transmission of the queue by clicking on the "transmit now" button. This can interrupt receiving messages on the Gateway from other stations for a while.

Stations Information

Manage the stations network, giving nicknames for each station and configuring the preferred transmission data frequency and mode for each connection or enable and disable connections.

Alias - Station identification.

Nickname - Name for easy or representative identification.

Status - Connection status between stations.

Frequency - Preferred digital data frequency for connection between stations.

Mode - Preferred mode for connection between stations (USB/LSB).

Wi-Fi & Ethernet manager

Wi-Fi & Ethernet manager, configure Wi-Fi SSID (network name), change password and check Wi-Fi and ethernet information.

*Note: Changing the SSID or the Wi-Fi password will disconnect everybody from the Wi-Fi network.

Troubleshooting

To know what to do in case of an error, you first need to know which component is causing the problem. To find out if the problem is with the device connecting to the network (cell phone, tablet, computer), the easiest way is to test access from another device, or from another browser on the same device. If the problem is only with you, it is possible that the problem is in the network settings of your device. Check below the necessary settings to access the HERMES network: If this does not solve your problem, try cleaning your browser's history and logging in again, or change browsers. If none of this resolves the problem, it is possible that your device is incompatible with the network. In case of problem in the Antenna, a LED will light up on the server box, in this case, check the correct position of the power supply wires and check the position and integrity of the Antenna, coaxial cable, and the transmitter. If the messages in the transmission list are not being transmit, check if the radio settings have been changed, in which case it could be that the other stations are offline or on a different frequency. To remedy you could restore the factory default settings. If the problem is with the digital radio, there will be an error message when the administrator goes into Administration -> radio configuration. If you receive a message saying server offline, your equipment is likely turned off. If the message persists, it could be that some part of the system is corrupted. In this case, you may need to take your equipment to a place with internet for online maintenance.

Media Access Control (MAC)

A Media Access Control (MAC) address, sometimes referred to as a hardware or physical address, is a unique, 12-character alphanumeric attribute that is used to identify individual electronic devices on a network. In this section you can enable or disable the MAC verification in reason of restrict the WiFi access. Add a new MAC address to provide access permission to the Wi-Fi HERMES for a unique device.

An example of a MAC address is: 00-B0-D0-63-C2-26.
